# Description
Ingame GSC timer for speedrunning BO2 easter egg maps. Automatic start, split, and stop functionality. Single script that works on all maps. Acurate times down to increments of 50ms.

## Supported runs  
  [✔️] Tranzit solo  
  [❌] Tranzit coop  
  [❌] Die rise  
  [✔️] MotD solo  
  [✔️] MotD coop  
  [✔️] Origins solo  
  [✔️] Origins coop  
  [❌] Buried
